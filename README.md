# 🛡️ ECHELON | AI Talent Screening

**Echelon** is a high-performance recruitment tool designed to automate the screening of unstructured resumes and structured talent profiles. Built for the **Umurava AI Hackathon (April 2026)**.

---

## 🚀 Project Structure
To maintain a clean codebase, the project is divided into:
* `/frontend`: Next.js application (User Interface).
* `/backend`: Node.js/Express server (AI Logic & APIs).
* `/shared`: Sample resumes, job descriptions, and mock JSON data.
* `/docs`: Technical guides and system architecture.

---

## 🛠️ Tech Stack (Planned)
* **Frontend**: Next.js, TypeScript, Tailwind CSS.
* **Backend**: Node.js, Express, MongoDB.
* **AI**: Google Gemini API (Pro/Flash).
* **Parsing**: Multer & PDF-Parse.

## 🔐 Contribution Policy
This repository is managed by the **Lead (Arsene)**. 
1. Team members work locally on their features.
2. Completed features are sent as ZIP files to the Lead.
3. The Lead integrates, tests, and pushes to the `dev` branch.
4. Stable versions are merged into `main`.
